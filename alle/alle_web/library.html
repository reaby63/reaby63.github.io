設計說明 Design Concept
圖書館管理系統，適用於中小型圖書館、圖書室，系統提供讀者自助借還書的功能，讓單位更能方便且快速管理書籍去向。我們亦提供了快速編目服務，掃描ISBN 系統即會帶出書目資料、館藏登錄號自動取號，讓您快速建立館藏。
功能特色 System Features

快速編目

掃描ISBN 即會帶出書目資料、館藏登錄號自動取號，快速建立館藏。

雲端服務

借閱紀錄將會儲存在雲端資料庫內，不用擔心資料遺失。

支援所有瀏覽器

此系統裝置適用於目前所有網頁瀏覽器，包括IE、Chrome、FireFox等。

環保無紙管理

僅需系統管理，免用紙張，友善環境。

借閱設備擴充

可加購自助借還書機，讀者可自行借還書。

提供學校、擁有小型圖書館的公司團體，

簡單方便好上手的圖書管理系統。

自助借閱無需人力

購書即可自動編目
